# Cloze Flashcards — Renal Mass Characterization

| Statements | Number |
|---|---|
| AML shows {{c1::negative}} HU on CT pre-contrast due to macroscopic fat. | 1 |
| RCC shows {{c1::heterogeneous, avid}} enhancement on CT. | 2 |
| Simple cysts show {{c1::no}} enhancement on CT. | 3 |
| Oncocytoma shows {{c1::homogeneous, avid}} enhancement with a central scar. | 4 |
| AML's key differentiating feature is {{c1::macroscopic fat}} on CT or chemical shift MRI. | 5 |
| RCC's key differentiating features include {{c1::pseudocapsule}}, tumor thrombus, and metastases. | 6 |
| Simple cysts have a {{c1::thin imperceptible wall}} with no septa. | 7 |
| Oncocytoma's key differentiating feature is a {{c1::central stellate scar}} on delayed imaging. | 8 |
| AML has {{c1::high}} hemorrhage risk (>4 cm; Wunderlich syndrome). | 9 |
| RCC has {{c1::moderate}} hemorrhage risk. | 10 |
| Simple cysts have {{c1::no}} hemorrhage risk. | 11 |
| Oncocytoma has {{c1::low}} hemorrhage risk. | 12 |
| RCC shows {{c1::20-70}} HU on CT pre-contrast (soft tissue). | 13 |
| Simple cysts show {{c1::0-20}} HU on CT pre-contrast (water density). | 14 |
| Oncocytoma shows {{c1::30-50}} HU on CT pre-contrast. | 15 |
| AML enhancement is {{c1::minimal}} (vascular smooth muscle). | 16 |
| AML on MRI T1 shows {{c1::high}} signal (fat). | 17 |
| RCC on MRI T1 shows {{c1::isointense to hypointense}} signal. | 18 |
| Simple cyst on MRI T1 shows {{c1::homogeneous low}} signal. | 19 |
| Oncocytoma on MRI T1 shows {{c1::isointense}} signal. | 20 |
| AML on MRI T2 shows {{c1::high}} signal with chemical shift. | 21 |
| RCC on MRI T2 shows {{c1::heterogeneous high}} signal. | 22 |
| Simple cyst on MRI T2 shows {{c1::homogeneous very high}} (CSF-like) signal. | 23 |
| Oncocytoma on MRI T2 shows {{c1::isointense to slightly high}} signal. | 24 |
| AML growth pattern is {{c1::slow and indolent}}. | 25 |
| RCC growth pattern is {{c1::variable}} (can be aggressive). | 26 |
| Simple cyst growth pattern is {{c1::none}}. | 27 |
| Oncocytoma growth pattern is {{c1::benign, may grow slowly}}. | 28 |
